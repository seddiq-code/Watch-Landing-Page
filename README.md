# Luxury Watch Landing Page

A sleek, fully responsive, and modern landing page designed to showcase premium timepieces. Built using clean, semantic HTML5, modern CSS3 transitions, and interactive vanilla JavaScript.

---

[✨ Live Demo](YOUR_GITHUB_PAGES_LINK_HERE)

---

##  Features

* **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
* **Interactive Gallery:** Smooth slider to view different watch models.
* **Modern UI/UX:** Clean typography, subtle animations, and premium color palettes.

---

##  Tech Stack

* **HTML5:** Semantic markup for optimal SEO and accessibility.
* **CSS3:** Custom properties (variables), Flexbox, Grid, and keyframe animations.
* **JavaScript (ES6+):** DOM manipulation, event listeners, and dynamic UI updates.

## Customization

To make this project your own, you can modify:
* `index.html`: Update the watch names, descriptions, and prices.
* `style.css`: Change the `:root` color variables to match your preferred branding.
* `/assets/images/`: Replace the placeholder images with your own high-resolution watch photography.
